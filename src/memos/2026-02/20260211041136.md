## **突然有一点后悔转专业了**  
  


（前几天写的，现在又不觉得后悔了）

  


我想学 graphics，也想从事 graphics 相关的工作。可是这个破 game programming 的课程怎么那么那么那么那么多的作业，每周一个 project，开学五周又要 midterm 了。这学期上半部分又一直沉迷于搞开源，一点东西没学鼠鼠要鼠掉了真是的。

  


**我讨厌使用 Windows** ，每次一打开 Windows 看到它丑陋的 ui 听到它刺耳提示声和无无无无无穷尽的配置问题我就心烦。我讨厌打开 visual studio ，可是这quarter两个老师都强制使用 vs 来写作业，写完了还没完，你需要录制一个 video report 上传到 YouTube 。你需要使用 perforce 而不是常见的 git 来管理代码，你需要用鼠标点点点，而不是键盘来自动化。**我讨厌 Windows**。

  


这个专业让我更多地接触到似乎是游戏工程的东西，可是我想学渲染想学动画想写 shader ，我不想知道 gpu 的缓冲区发生了什么，不想知道不同的 filtering 不同的 mipmap 有什么区别。我对 graphics 言语上的热爱仿佛是假的。

  


啊，重点还是，**我讨厌 Windows** 。我所有的痛苦是不是都是强制使用 Windows 强制用 DX11 API 来完成作业造成的？ （我现在来回答，是的是的是的，不用 Windows 不打开 Visual Studio 我的人生快乐十倍！！）
